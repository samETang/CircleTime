# CircleTime

**CircleTime** is a full-stack classroom management ecosystem designed to bridge the gap between home and school. Built specifically for the unique needs of Early Childhood Education (ECE) and inclusive classrooms, it prioritizes professional boundaries, developmental tracking, and multi-stakeholder coordination.

## 🚀 The Vision
In an era of constant connectivity, educators often face burnout due to 24/7 communication expectations. CircleTime reimagines this dynamic by implementing "Boundary-First" logic while providing parents and specialists with the deep, data-driven insights they need for student success.

## ✨ Key Features
* **Boundary-First Messaging:** A custom communication interface that honors "Contract Hours." After-hours messages are automatically queued for the next school day, ensuring teachers can disconnect without missing important updates.
* **Support Circles:** Dedicated, secure sub-channels for students with IEPs or 504 plans. This allows teachers, parents, and specialists (OTs, SLPs, etc.) to coordinate in a unified thread.
* **Developmental Milestone Engine:** Moves beyond binary "behavior points" to track social, emotional, and physical growth markers through a rich, visual daily log.
* **Document Certification:** A secure repository for sharing and verifying specialist reports and developmental certifications.

## 🛠️ Technical Stack
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** PostgreSQL (for relational support team architecture)
* **Real-Time:** Socket.io for Support Circle synchronization
* **Authentication:** JWT-based secure role-based access control (RBAC)

## 📂 Architecture Highlights
* **Role-Based Access Control:** Granular permissions ensuring specialists only access data for assigned students.
* **Time-Fencing Logic:** Middleware-driven messaging controls based on educator-defined professional hours.
* **Relational Schema:** Optimized PostgreSQL structure to handle complex many-to-many relationships between students, teachers, and external specialists.
